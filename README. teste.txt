TESTE NERDWEB
Bom Dia.

1. Eu tinha a intenção de componentizar o HTML, porém não encontrei a forma de fazer isso na internet. Eu conseguiria fazer isso no React, mas como não era para utilizar nenhum Framework optei por desenvolver todo html em um arquivo único;

2. O scss foi onde produzi todo o css do arquivo html, inclusive comentei cada "componente" no arquivo para ficar mais fácil a localização de cada estilo;

3. Criei duas funções com o Javascript. 
    1. A primeira foi a de clicar na seta que está no topo da página no menu para abrir a div oculta.
    2. A segunda foi a validação dos inputs Name e Email e o armazenamento dos dados no localstorage.

4. Criei a responsividade em três modelos:
    1. WEB: Padrão, Width 1920px;
    2. Notbook: Max-width: 1024px;
    3. Mobile: Max-width: 666px;

5. Os links do final da página das redes sociais levam para as páginas da Nerdweb (exceto o twitter que não encontrei);

6. Os demais botões não adicionei nenhuma função pois provavelmente seriam rotas para outras páginas;
